Python Intro Notes

Sat, 15 May 2021

My notes from "An Informal Introduction to Python"

tags: python - beginner

Numbers

  • / returns a float by default
>>> 17 / 3  # classic division returns a float
5.666666666666667
  • // will return the floor-rounded result of division (essentially discards the decimal)
>>> 17 // 3  # floor division discards the fractional part
5
  • Mixing number "types" is allowed
>>> 4 * 3.75 - 1
14.0

Strings

  • Strings can be concatenated using n * <string>
>>> 3 * 'un' + 'ium'
'unununium'
  • Adjacent string literals are auto-concatenated
>>> 'Py' 'thon'
'Python'
  • Access string index ranges using the [x:y] syntax
>>> word = 'Python'
>>> word[0:2]
'Py'
>>> word[2:5]
'tho'
